### To start the docker:
```
sudo docker-compose up -d
```

### NodeRed:
On the browser enter 
```
https://server-ip:1880
```

### Grafana:
On the browser enter 
```
https://server-ip:3000
```

### Change Username:
Change the username on the file .env
By default, it was
```
username: admin
password: admin
```
